# atributos
- placeholder
- readonly / disabled
- value
- required * minlenght / maxlength
- O mínimo/ máximo valor que o campo irá conter
* size
- valor númerico identificando quantos caracteres o campo deverá conter
* pattern
- uso de expressão regular para validar o campo - exemplo: pattern=".\.com\.br\/.*"
* list
- o id de ua tag datalist que está no mesmo documento
- datalist irá conter uma lista de valores pré definidos
- só apresenta valores compatíveis com o campo
* spellcheck
- habilita a verificação ortofráfica para este input
