- Usuário poderá escolher 1 ou mais arquivos para enviar
# Atributos
- value - contém o arquivo a ser enviado
* accept
- descreve quais tipos de arquivos são aceitos
- exemplo: .doc, .docx, .pdf, audio/*, image/png, .png
- files - a lista de arquivo ou arquivos
- multiple - envio de múltiplos arquivos

# Envio dos arquivos
- para envio dos arquivos o formulário deverá ultilizar o método POST e o atributo enctype como "multipart/form-data".
