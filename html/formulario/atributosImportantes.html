Autocomplete completa conforme as informações mais usadas no navegador

autofocus dar foco ao input, somente um por página

disabled deixa o input desabilitado

já vem com valor definido, se acescentar disabled não é possível mexer

a mesma coisa que o disabled só muda o aspecto

atributo form serve para linkar com algum formulário

required define que é obrigatório responder o input

placeholder deixa uma mensagem dentro do input



obs: nem todos os atributos funcionam com todo tipo de campo
